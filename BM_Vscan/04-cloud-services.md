# 4. 디지털/클라우드 생태계 (구독형 부가서비스)

GE가 "Digital Tools for Vscan Air"로 묶어 제공하는 3종 서비스 (기업용/개인용 요금제 별도):

1. **MyDeviceHub**: IT 관리자용 웹 기반 중앙관리 콘솔. 부서별 기기 그룹핑, 사용자 권한관리, 설정 배포, 사용 현황 추적 (fleet management).
2. **MyRemoteShare** (Powered by Zoom): 스캔 화면 실시간 공유 + 음성/영상 통화. 원격 트레이닝, 초음파 초심자 지도, 동료 자문에 사용. 일본어 자료에서 "리얼타임 에코 화면 공유" 기능으로 강조.
3. **MyImageCloud**: 클라우드 PACS 저장/리뷰 솔루션. DICOMweb export 지원. 개인 사용자용(엔터프라이즈 아님), 연 500 exam 제한, 데이터는 구독 유지 시 최대 7년 보관, 구독 해지 시 30일 내 다운로드 필요.

이 구독 모델은 Fierce Biotech 등에서 "digital tool subscriptions" 출시로 보도됨.
