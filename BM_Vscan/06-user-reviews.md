# 6. 사용자 리뷰/피드백 (UI·UX 실사용 관점)

## 긍정적

- "300회 이상 스캔했다"는 리뷰 등 실사용 만족도 존재, annotation 기능은 이후 업데이트로 추가되어 개선됨.

## 부정적/문제점 (실제 개발 시 참고할 만한 부분)

- **iOS 18 초기 크래시** 이슈 보고 (몇 분 사용 후 앱 강제종료).
- **UI 학습곡선**: "UI takes some getting used to but has great features if you explore deep enough" — 즉 스와이프 기반 좌/우 패널 등 제스처 UX가 처음엔 직관적이지 않다는 평.
- **연결 안정성**: Wi-Fi Direct 연결이 약 90% 신뢰성, 절반 정도는 연결이 느림. Exam당 1~2회 정도 신호가 순간적으로 끊기며 화면이 얼었다가 좌상단에 "low signal" 표시가 뜨는 패턴이 반복적으로 보고됨. → 실시간 무선 스트리밍 앱을 만든다면 재연결/신호품질 UI 피드백을 신경 써야 할 지점.
- 과거 버전에서는 이미지 주석(annotation)/텍스트 메모 기능 부재가 큰 불만이었음(추후 패치로 해결).
- 계정 잠금(lockout) 이슈로 응급 상황에서 로그인이 막히는 문제 → Guest mode가 이 문제의 대응책으로 보임.
